U -> View: Finaliza un contenido o actividad
View -> Api: recordActivityCompletion(activityData)

alt #FFFDD0 No hay conexión a internet
    Api -> Api: Almacenar datos localmente (LocalStorage/IndexedDB)
    note right: Los datos se enviarán cuando\nse recupere la conexión
end

Api -> Ctrl: POST /api/activities/complete (dto)
Ctrl -> Srv: saveCompletedActivity(dto)

Srv -> Srv: Validar que el usuario sea Alumno o Externo

Srv -> Repo: save(ActivityRecord)
Repo -> DB: INSERT INTO activity (date, start_time, duration, activity_id)
DB --> Repo: confirmation
Repo --> Srv: ActivityRecord
Srv --> Ctrl: success
Ctrl --> Api: 201 Created

Api --> View: notifySuccess
View --> U: Muestra confirmación de actividad guardada
