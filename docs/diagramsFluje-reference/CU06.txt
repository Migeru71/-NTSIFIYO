@startuml
skinparam sequence {
    LifeLineBackgroundColor #F5F5F5
    ParticipantBackgroundColor #FFFDD0
    ParticipantBorderColor #666666
}
skinparam NoteBackgroundColor #F5F5F5

title Visualizar contenido audiovisual con subtítulos

actor "Usuario" as U
boundary "ContenidoView" as View
control "ApiService" as Api
control "ContentController" as Ctrl
entity "ContentService" as Srv
entity "ContentRepository" as Repo
database "MySQL" as DB
entity "CloudStorage" as Cloud

autonumber

U -> View: Entra a página de contenido
View -> Api: fetchContent(filters: season, grade, difficulty)
Api -> Ctrl: GET /api/content?filters...
Ctrl -> Srv: getPaginatedContent(filters)

group Búsqueda y Filtrado
    Srv -> Repo: findByFilters(filters)
    Repo -> DB: SELECT * FROM media WHERE ...
    DB --> Repo: List<Media>
    Repo --> Srv: List<MediaEntity>
    
    alt #FFFDD0 No hay contenido para los filtros seleccionados
        Srv --> Ctrl: throw NoContentException
        Ctrl --> Api: 404 Not Found
        Api --> View: Muestra estado vacío
        View --> U: Mostrar: "No se encontró contenido disponible"
    end
end

Srv --> Ctrl: Page<Media> (Leyendas, anécdotas, canciones)
Ctrl --> Api: 200 OK
Api --> View: Renderizar lista seccionada

U -> View: Selecciona un elemento
View -> Api: getContentDetails(id)
Api -> Ctrl: GET /api/content/{id}
Ctrl -> Srv: getDetails(id)
Srv -> Repo: findById(id)
Repo -> DB: SELECT * FROM media WHERE id=?
DB --> Repo: MediaEntity
Repo --> Srv: MediaEntity
Srv --> Ctrl: MediaDetailsDto
Ctrl --> Api: 200 OK
Api --> View: Mostrar descripción y vista previa

U -> View: Confirma reproducción
View -> Api: getStreamResources(id)
Api -> Ctrl: GET /api/content/{id}/stream

group Preparación de Recursos Multimedia
    Ctrl -> Srv: prepareStreamData(id)
    Srv -> Repo: findAllSubtitlesByMediaId(id)
    Repo -> DB: SELECT * FROM subtitles WHERE media_id=?
    DB --> Repo: List<Subtitle>
    Repo --> Srv: List<Subtitle>
    
    Srv -> Cloud: generateSignedUrl(filePath)
    note right: Genera URL temporal para acceso seguro
    Cloud --> Srv: signedUrl
    
    Srv --> Ctrl: StreamResourcesDto (URL + Subtitles)
end

Ctrl --> Api: 200 OK (Data Block)
Api --> View: Load Video Player with Subtitles

View -> View: Reproducción HSL con subtítulos sincronizados (ESP/MAZ)
note right: Los subtítulos se cargan completos al inicio
View -> U: Mostrar aviso de fin de reproducción

@enduml
